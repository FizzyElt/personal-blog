Effect-TS 所主打的 `Effect` Module 對前端來說還沒有一個很好的使用情境，與前端框架難以融合的很好，所以在前端使用 Effect-TS 的主要關注點就在於不與 `Effect` 掛勾能夠獨立運作的 Module，因此大概瀏覽了整個 API 文件後整理出以下清單提供參考。

- `String`
- `BigDecimal`
- `BigInt`
- `Boolean`
- `Number`

- `Array`
- `Record`
- `Struct`
- `Trie`
- `Tuple`
- `HashSet`
- `MutableHashSet`
- `SortedSet`
- `HashMap`
- `MutableHashMap`
- `SortedMap`
- `List`
- `MutableList`
- `MutableQueue`
- `MutableRef`
- `RedBlackTree`

### 比較相關

- `Equal`
- `Equivalence`

- `Ordering`

- `Either`
- `Option`
